Diseño Web Profesional,Practico y desde cero

Bienvenido al Curso de Diseño Web, curso en el que aprenderás paso por paso y desde lo más básico a diseñar cualquier página web de una forma práctica y sencilla.

No importa si nunca antes has programado o si no tienes conocimientos de páginas web. Este curso está diseñado para que aprendas desde lo más básico hasta un nivel avanzado en el que podrás diseñar cualquier sitio web de forma profesional.

¿Qué incluye este curso?

  • Más de 15 horas en video en las que aprenderás más de 150 temas diferentes.
  • 10 Proyectos en los cuales practicar todos los conocimientos aprendidos.
  • Garantía de 30 días

Aprenderás:

  • Crear diseñar página web profesional desde cero con HTML y CSS
  • Manipular tus sitios web para agregarles animaciones y efectos con CSS y jQuery
  • Convertir tú sitio web en un sitio adaptable a dispositivos móviles.
  • Programar en JavaScript uno de los lenguajes de programación más importantes.
